# Armorer Labs

### Run any agent. Securely.

Armorer Labs builds local-first infrastructure for safer AI agents: sandboxed runtimes, guided setup, credential handling, human approvals, guardrails, and audit visibility.

## What We Build

- **Armorer**: a secure local control plane for installing, configuring, running, and monitoring AI agents.
- **Armorer Guard**: local guardrails for credential leakage, suspicious requests, and risky agent actions.
- **Evaluation tooling**: repeatable safety and runtime tests before changes ship.

## Why It Exists

Agents are becoming useful enough to run real workflows, but direct execution is still too messy: credentials scatter, logs disappear, permissions are implicit, and risky actions are hard to review.

Armorer Labs makes the safe path the easy path: local by default, observable by design, and built for builders who want control before scale.
